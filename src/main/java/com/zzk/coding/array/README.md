### 题目
给定一个数组arr，该数组无序，但每个值均为正数，再给定一个数组k。求arr的所有子数组中所有元素相加和为k的最长子数组长度
例如：arr=[1,2,1,1,1], k=3
累加和为3的最长子数组为[1,1,1]，所以结果返回3

## 12.未排序数组中累加和小于或者等于给定值的最长子数组长度

例如：arr=[3,-2,-4,0,6]，k=-2，相加和小于或等于-2的最长子数组为{3, -2, -4, 0}，所以结果返回4
